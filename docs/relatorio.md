## Principais Comandos Git Utilizados
git init
Inicializa um novo repositório Git numa pasta.

git clone <url>
Cria uma cópia local de um repositório remoto.

git status
Mostra o estado atual dos ficheiros (alterados, prontos para commit, etc.).

git add .
Adiciona todos os ficheiros modificados à área de preparação (staging).

git commit -m "mensagem"
Guarda as alterações com uma mensagem descritiva.

git log
Apresenta o histórico de commits do repositório.

git branch
Lista as branches existentes.

git branch nome-branch
Cria uma nova branch.

git checkout nome-branch
Muda para outra branch.

git checkout -b nome-branch
Cria e muda automaticamente para uma nova branch.

git merge nome-branch
Junta uma branch à branch atual.

git pull
Atualiza o repositório local com alterações do repositório remoto.

git push
Envia os commits locais para o repositório remoto.

git remote -v
Mostra os repositórios remotos configurados.

git diff
Mostra as diferenças entre versões dos ficheiros.
